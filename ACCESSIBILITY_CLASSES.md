# Классы для режима доступности

Для оптимального управления элементами в режиме для слабовидящих была создана система CSS классов, которые автоматически применяются только когда активен режим доступности.

## Основные классы

### 1. `accessibility-hide`
**Назначение:** Полное скрытие декоративных элементов
**Применение:** Иконки, декоративные изображения, элементы дизайна

```html
<!-- Декоративная иконка, которая не несет смысловой нагрузки -->
<i class="fas fa-star accessibility-hide"></i>

<!-- Декоративный элемент -->
<div class="decorative-element accessibility-hide"></div>
```

### 2. `accessibility-hide-animation`
**Назначение:** Отключение анимаций и переходов
**Применение:** Слайдеры, анимированные элементы, эффекты наведения

```html
<!-- Слайдер с анимацией -->
<div class="slider accessibility-hide-animation">
    <!-- содержимое слайдера -->
</div>

<!-- Кнопка с эффектом наведения -->
<button class="btn-animated accessibility-hide-animation">Кнопка</button>
```

### 3. `accessibility-hide-bg`
**Назначение:** Скрытие фоновых изображений
**Применение:** Элементы с декоративными фонами

```html
<!-- Секция с фоновым изображением -->
<section class="hero-section accessibility-hide-bg">
    <h1>Заголовок</h1>
    <p>Текст остается видимым</p>
</section>
```

### 4. `accessibility-hide-interactive`
**Назначение:** Отключение сложных интерактивных элементов
**Применение:** Сложные виджеты, интерактивные карты, игры

```html
<!-- Интерактивная карта -->
<div class="interactive-map accessibility-hide-interactive">
    <!-- карта становится неактивной и полупрозрачной -->
</div>
```

### 5. `accessibility-simplify`
**Назначение:** Упрощение визуального оформления
**Применение:** Карточки, блоки с тенями, скругленные элементы

```html
<!-- Карточка с тенью и скруглением -->
<div class="card shadow-lg rounded-lg accessibility-simplify">
    <h3>Заголовок карточки</h3>
    <p>Содержимое карточки</p>
</div>
```

## Рекомендации по использованию

### Что скрывать:
- ✅ Декоративные иконки и изображения
- ✅ Анимации и переходы
- ✅ Сложные интерактивные элементы
- ✅ Фоновые изображения
- ✅ Элементы, дублирующие информацию

### Что НЕ скрывать:
- ❌ Информативные изображения
- ❌ Навигационные элементы
- ❌ Формы и кнопки
- ❌ Основной контент
- ❌ Альтернативный текст

## Примеры применения

### Карточка врача
```html
<div class="doctor-card accessibility-simplify">
    <!-- Декоративная иконка - скрываем -->
    <i class="fas fa-user-md accessibility-hide"></i>
    
    <!-- Фото врача - оставляем, но можем упростить фон -->
    <img src="doctor.jpg" alt="Доктор Иванов" class="accessibility-hide-bg">
    
    <!-- Основная информация - оставляем -->
    <h3>Доктор Иванов</h3>
    <p>Офтальмолог</p>
    
    <!-- Анимированная кнопка - убираем анимацию -->
    <button class="btn-primary accessibility-hide-animation">Записаться</button>
</div>
```

### Слайдер преимуществ
```html
<div class="advantages-slider accessibility-hide-animation">
    <!-- Декоративные стрелки - скрываем -->
    <button class="slider-arrow accessibility-hide">←</button>
    
    <div class="slider-content">
        <!-- Контент остается видимым -->
        <h3>Преимущество 1</h3>
        <p>Описание преимущества</p>
    </div>
    
    <button class="slider-arrow accessibility-hide">→</button>
</div>
```

## Автоматическое применение

Все классы работают автоматически только в режиме доступности. Когда пользователь включает версию для слабовидящих, элементы с этими классами автоматически скрываются или упрощаются.

Никаких дополнительных JavaScript действий не требуется - достаточно добавить соответствующий класс к элементу.
